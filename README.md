LES COEFFICIENTS DE BEZOUT sont deux entiers x et y qui vérifient l'égalitè suivant :
a*x+b*y= PGCD(a,b).
POUR deux entiers a et b le théoréme de bézout affirme qu'il existe toujour deux entiers x et y pour que cette équation soit vraie.
Pour trouver les coefficient de bézout en utilise l'algorithme d'euclide étundu :
1- on utilise algorithme d'euclide pour calculer pgcd.
2- on remontre le calcule pour exprimer le pgcd comme combinaison linéaire de a et b.
